# SmithersXServer

Simple X11 Server

Not yet working

## How to compile and run

$ perl build.pl

To run the X Server (runs on port 6002):

$ ./smithers XServer

Then run a program that connects to port 6002 (:2 means 6000+2):

$ DISPLAY=:2 xpenguins
